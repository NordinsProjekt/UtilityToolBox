# UtilityToolBox Examples

This directory contains example projects demonstrating how to use the UtilityToolBox.Services library.

## UtilityToolBox

A console application demonstrating menu creation and display with both horizontal and vertical orientations.

### Running the Example

```bash
cd examples/UtilityToolBox
dotnet run
```

### Example Code

The example shows how to:
- Create menus using `MenuFactory`
- Add menu items using fluent API
- Automatically generate menu options
- Display menus in different directions

See the files in `Menu/` directory for specific examples:
- `HorizontalMainMenu.cs` - Shows horizontal menu layout
- `VerticalMainMenu.cs` - Shows vertical menu layout
